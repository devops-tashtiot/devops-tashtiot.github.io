🔄 aws-xray-db-sync

Lambda function (Python 3.12) that runs every minute via EventBridge Scheduler to download JFrog Xray offline vulnerability DB bundles into S3. Infrastructure managed with Terraform + Terragrunt. Sends SNS email alerts on success or failure.

AWS Lambda EventBridge S3 Terraform Terragrunt
🏗️ aws-terraform-bootstrap

One-time Terraform that provisions the shared S3 bucket used for remote state across all projects in the AWS account. Run once per account — every other repo depends on this bucket for Terragrunt state storage.

Terraform S3 Remote State Bootstrap
📋 rhdh-templates

Red Hat Developer Hub (RHDH) scaffolder templates for the tashtiot platform. Used by developers to bootstrap new services and projects through a self-service internal developer portal.

RHDH Backstage Scaffolding IDP
⚡ example-api

FastAPI reference template demonstrating the standard project structure for platform APIs. Includes DNS and FQDN service endpoints as examples — intended as a starting point for new API projects.
