🏗️ aws-terraform-bootstrap

One-time Terraform that provisions the shared S3 bucket used for remote state across all projects in the AWS account. Every other repo depends on this bucket for Terragrunt state storage.

🔄 aws-xray-db-sync

Lambda function (Python 3.12) that runs every minute via EventBridge Scheduler to download JFrog Xray offline vulnerability DB bundles into S3. Sends SNS email alerts on success or failure. Infrastructure managed with Terraform + Terragrunt.
